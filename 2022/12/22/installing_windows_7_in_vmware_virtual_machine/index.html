Installing Windows 7 in a VMware Virtual Machine

Installing Windows 7 in a VMware Virtual Machine

1. Installing VMware Virtual Machine

  • Reference: Installing VMware Virtual Machine

2. Creating the Virtual Machine

  1. Click Create a New Virtual Machine on the main interface.

  2. Select Typical, then click Next.

  3. Here, we choose Install the operating system later. If we selected the system image here, VMware would automatically complete the system deployment for us. However, the system would be in poor condition after deployment, so we will manually deploy it.

  4. Select Windows 7 for the operating system.

  5. Set the name and storage location for the virtual machine. It is strongly recommended to place the virtual machine in a disk other than the C drive.

  6. Enter the disk capacity based on your needs. Generally, it should not be less than 20GB to reserve space for the system.

  7. Click Customize Hardware at this step.

  8. Find the New CD/DVD option, click Use ISO image file, and then select your ISO image file. Here is the download link for Windows 7 64-bit Professional with sp1 Batch Version. It is recommended to use Xunlei for downloading (make sure to select the version with vl, otherwise activation will be a nightmare).

  9. For the Network Adapter, it is recommended to change it to Bridged Mode. However, it won’t affect usage if you don’t change it.

  10. After closing the Hardware Settings page, click Finish.

  11. Go to the newly created virtual machine and click Power on this virtual machine.

  12. Wait for the virtual machine to start.

2. Installing the Operating System

  1. Select the language you understand, then click Next – Install Now.

  2. Accept the license terms.

  3. Choose Custom (Advanced).

  4. On the disk selection page, click Drive Options (Advanced) (A).

  5. Select the unallocated disk, then click New – Apply – OK.

  6. Click Next, and the system will install automatically. You can use this time to enjoy a coffee.

  7. After several restarts, the virtual machine will enter the system setup page. Set a username you like, then click Next all the way through. You can choose to set or not set a password. Updates will be discussed later; just follow the images for the rest.

  8. Enter the operating system.

3. Installing VMware Tools

  1. Click Virtual Machine – Install VMware Tools in the menu bar.

  2. If autoplay prompts a window, click Run. If no window pops up, find the DVD drive in the computer and double-click it.

  3. During the installation process, just click Next all the way through.

  4. If there are driver issues, select Always install this driver software.

  5. If you encounter the following error, refer to “Solving Windows 7 VMware Tools Installation Errors”.

  6. After installation, restart the system once.

  7. The benefit of installing VMware Tools is that the resolution can adapt automatically, and you can directly drag and drop files into the virtual machine.
