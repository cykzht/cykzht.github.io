1.2.2 语法规则

switch case 语句有如下规则：

  • switch 语句中的表达式结果的类型可以是： byte、short、int 或者 char。从 JDK1.5 开始，支持 enum，从 Java SE 7 开始，支持字符串 String 类型
  • switch 语句可以拥有多个 case 语句。每个 case 后面跟一个要比较的值和冒号。
  • case 语句中的值的数据类型必须与表达式结果的数据类型相同，而且只能是常量或者字面常量。
  • 当表达式的值与 case 语句的值相等时，那么 case 语句之后的语句开始执行，直到 break 语句出现才会跳出 switch 语句。
  • 当遇到 break 语句时，switch 语句终止。程序跳转到 switch 语句后面的语句执行。case 语句中 break 是可选的。如果没有 break 语句出现，程序会继续执行下一条 case 语句，直到出现 break 语句或者 switch 语句执行完毕。
  • switch 语句可以包含一个 default 分支，该分支一般是 switch 语句的最后一个分支（可以在任何位置，但建议在最后一个）。default 在没有 case 语句的值和表达式的值相等的时候执行。default 分支不需要 break 语句。
