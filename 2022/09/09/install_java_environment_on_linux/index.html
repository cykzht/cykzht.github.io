Installing Java Environment on Linux

JDK is the software development kit for the Java language, and it is the core of Java development. It includes the Java runtime environment (JVM + Java system libraries) and Java development tools.
This article will install Java 19 on CentOS7.

3.1 JDK Download

  • Search for Java in Baidu, then click to enter the Java | Oracle official website (make sure not to click the wrong one).

  • Click on Developer Resources in the top menu of the official website.

  • Click on Download at the top of the new page.

  • On the new page, click Download Java at Oracle.com.

  • Scroll down to find Java19, select the Linux version (if you are using a different operating system, please select the corresponding version), find x64 RPM Package, and copy the download link on the right.

  • Open the terminal on Linux or connect to Linux via SSH.

  • Use the wget command to download the package. If you don’t have wget, use the following command to install it.

    Ubuntu

    CentOS

3.2 JDK Installation

  • Use the rpm command to install Java.

    • -i means install, which installs the software.
    • -v displays detailed information during the installation process.
    • -h shows the installation progress with # symbols.

  • You can use it directly next.

3.3 JDK Uninstallation

  • Use the rpm command to uninstall the package.
