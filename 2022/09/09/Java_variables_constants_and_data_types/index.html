Variables, Constants, and Data Types in Java

发表于2022-09-09|更新于2025-08-07|英文期刊

I. Variables

In Java, a variable refers to a quantity whose value can change during program execution. It is often used to store user input information, intermediate results of program execution, etc. Essentially, it is a storage area in memory used to store data.

  • A storage area in memory
  • The data in this area can change continuously within the same type range
  • Variables are the most basic storage units in a program

1.1 Defining Variables

In Java, variables are declared in the following format:

  • Declaring a single variable
    variableType variableName;
  • Declaring multiple variables
    variableType variableName1, variableName2, variableName3, ...;

Java does not allow the use of undeclared or declared but uninitialized variables, so when a variable has not been assigned an initial value or needs to be changed again, it is necessary to use an assignment statement.
variableName = variableValue;

Java can assign an initial value to a variable at the same time as declaring it

  • Initializing the value of a variable at the time of declaration
    variableType variableName = initialValue;
  • Initializing the values of multiple variables at the same time
    variableType variableName1 = initialValue1, variableName2 = initialValue2, variableName3 = initialValue3, ...;

For a declared and assigned variable, you can directly access the value of the variable through its name.

1.2 Usage Principles

  • Declare first, use later
  • Access the content stored in the variable through the variable name
  • Scope of a variable (scope): the pair of {} where it is defined
  • Variables are only valid within their scope
  • Within the same scope, you cannot define variables with the same name

1.3 Variable Types

Java requires that the specific data type of a variable be specified at the time of declaration (a strongly typed language). Different data types determine how much storage space is allocated in memory.

Java Data Types

  • Basic Data Types

    • Numeric Types

      • Integer Types

        • byte: Represents an integer using one byte. Range (-128 ~ 127)

        • short: Represents an integer using two bytes. Range (-32,768 ~ 32,767)

        • int: Represents an integer using four bytes. Range (-2,147,483,648 ~ 2,147,483,647)

        • long: Represents an integer using eight bytes. Range (-9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807)

          The literal of a long type must end with L or l
          In actual development, int is usually used to define an integer variable.

      • Floating-Point Types

        • float: Single precision, represents a floating-point number using four bytes

        • double: Double precision, represents a floating-point number using eight bytes

          The literal of a float type must end with F or f
          In actual development, double is usually used to define a floating-point variable

    • Character Type: Represents a character using two bytes

      • char

        The literal of a char type must use a pair of single quotes as delimiters, and it must contain exactly one character
        The char type also includes escape characters (characters with special meanings that are difficult to express in normal ways, such as carriage return, newline, etc., starting with \ followed by one character to represent a specific escape character. For example ‘\r’,‘\n’,‘\t’,‘'’,‘"’
        The range of char is ‘\u0000’~‘\uffff’

    • Boolean Type

      • boolean

        Its value can only be true or false
        It is often used as the result of a logical expression in conditional statements, loops, etc.

  • Reference Data Types

    • Class (class)

      A string is not a basic type but is declared using the String class

      • String: A sequence of characters enclosed in a pair of "".

        • String

          Whether it is a string literal or a string variable, it is implemented using a class. There are three classes in Java used to represent strings:

          • Immutable character sequence
            • String ([Detailed information](#IV.String Type String))
          • Mutable character sequence
            • StringBuilder
            • StringBuffer

    • Interface (interface)

    • Array ([])

      A collection of several elements of the same data type arranged in a certain order is called an array. These elements share a unified array name and are distinguished and managed through different indices.

II. Constants

A quantity whose value does not change during program execution is called a constant.
Its declaration method is similar to that of variables. Just add the keyword final in front.

It is recommended to provide a value for the constant at the time of declaration.

III. Data Type Conversion

The operation rules between basic data types.
Prerequisite: The discussion here only involves the operation between the other seven basic data types except for the boolean type.

3.1 Automatic Type Promotion

When a type with a smaller range of numbers is operated with a type with a larger range of numbers, the result is automatically promoted to the type with a larger range of numbers.
Particularly, when byte, char, short are operated, the result is of type int.
byte, short, char → int → long → float → double

When character data is operated with numeric data, it will first convert the value of the character data to the corresponding numeric data according to its encoding before performing the operation.

3.2 Forced Type Conversion

The inverse operation of automatic type promotion

  1. Explicitly specify the forced conversion type (desired conversion type) variableName
  2. Forced type conversion may result in loss of precision

If it is an integer, its default type is int. Long type data needs to end with L or l. Otherwise, the system will treat it as an int.

IV. String Type String

  1. String is a reference data type used to represent a sequence of characters (a string).
  2. String data of type must be enclosed in a pair of double quotes.
  3. String can perform operations with the eight basic data types, and can only perform concatenation operations (+).
  4. The result of the operation between String and other types of data is still of type String

The plus sign (+) in Java can represent both addition operations and string concatenation. If one of the values on both sides of the plus sign is of type String, it represents a concatenation operation. In other cases, it represents an addition operation.

Conversion between String type and basic data types

  • String to basic data type
    Use the parse method of the wrapper class of the basic data type
  • Basic data type to String
    Perform a concatenation operation with “”
