Java Development Environment JDK Installation Tutorial

JDK is the software development kit for the Java language. It is the core of Java development, containing the Java runtime environment (JVM + Java system libraries) and Java development tools. This article uses Java 19.

2.1 JDK Download

  • Search for Java in Baidu, then click to enter the Java | Oracle official website (make sure not to click the wrong one).

  • Click Developer Resources in the top menu of the official website.

  • Click Download at the top of the new page.

  • On the new page, click Download Java at Oracle.com.

  • Scroll down to find Java 19, select the Windows version (if you are using a different operating system, please select the corresponding version), find x64 Installer, and click the link to download.

2.2 JDK Installation

You can proceed with the installation by clicking “Next” all the way through (it is recommended not to install on the system drive C: and avoid using Chinese characters in the installation path).

2.3 Configure Environment Variables

Taking Win10 as an example.

  • In the taskbar search box, type environment variables, and click Edit system environment variables in the search results.
  • In the window that opens, click the environment variables button as shown in the red box.
  • In the window that opens, create a system variable JAVA_HOME


    Click OK to close the window after creating it.
  • Find the Path variable in the system, and edit this system variable as shown in the figure below:

2.4 Check if the Development Environment is Configured Successfully

  • In the taskbar search box, type cmd, and click Command Prompt in the search results.
  • In the command window, type java -version and press Enter. If it displays the JDK version information, the configuration is successful (the illustrated version is Java 1.8, but different versions are largely similar).
