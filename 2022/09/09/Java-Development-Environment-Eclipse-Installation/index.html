Installing the Java Development Environment Eclipse

发表于2022-09-09|更新于2025-08-07|英文期刊
|总字数:422|阅读时长:2分钟|浏览量:

I. About IDE

IDE stands for Integrated Development Environment, which integrates the tools or features you need during the development process, such as code writing, analysis, compilation, debugging, and more, thereby maximizing the developer’s work efficiency.

For Java environment configuration, please refer to Java Development Environment JDK Installation Tutorial (g2022cyk.top)

II. Downloading Eclipse

  1. Search for eclipse in baidu and go to the official download page Eclipse Downloads | The Eclipse Foundation. Do not enter other third-party websites.

  2. The official download page is shown in the figure below. Click on the Download Packages as indicated by the red box to download the unpacked version. This article is entirely based on the unpacked version.

  3. On the page that opens, select the corresponding Eclipse IDE for Java Developers version based on your operating system. Here, the author uses the Windows system, so download the corresponding Windows version.

III. Using Eclipse

  1. After downloading, it is a compressed package. We unpack it.

  2. Find the eclipse.exe file and double-click to open it.

  3. The first startup will ask you to configure the workspace directory. It is recommended to choose an empty folder outside the C drive.

  4. After opening, we create a new Java project by clicking File——New——Java Project in the menu bar to create the project.

  5. In this step, we need to give the entire program project a package name, and then we need to configure the Java runtime environment. The prerequisite is that your computer must have Java installed. For the Java installation tutorial, you can refer to the article Java Development Environment JDK Installation Tutorial.

  6. On the environment configuration page, click Add to add the installed Java environment.

  7. Select Standard VM.

  8. Find the directory where you previously installed Java and select this folder.

  9. After selecting the folder, click Finish to complete. Do not change anything else.

  10. Select the Java environment you just created and click Apply and Close.

  11. If an error occurs, it is due to the Java interpreter version (because different versions of Java have different writing methods, and the compiler needs to provide appropriate code hints based on your Java version). Click Configure... to modify it.

  12. In the Compiler compliance level, select your computer’s Java version. Here, the author uses Java18.

  13. After creating the project, you can start writing code. Create a new Java class in the src directory. The specific path is src(右键)——New——class to create a Java class.

  14. Give your code a name and click Finish to complete the creation.

  15. Start writing code.
