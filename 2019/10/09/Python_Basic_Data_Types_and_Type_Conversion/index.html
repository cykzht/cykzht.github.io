Basic Data Types and Type Casting in Python

Data Types

str String

Let’s get to know strings. The English word for string is string.

A string, as the name suggests, is a combination of characters strung together. Characters can be a number, a letter, a character, or even a symbol. Strings can express words, sentences, expressions, and more in the real world.

String recognition is very simple—it has a layer of skin called 【quotes】. As long as the content is enclosed by 【single/double/triple quotes】, regardless of whether the content is Chinese, English, numbers, or even Martian, as long as it is enclosed, it is considered a string type.

In this code snippet, 'world' and "HelloWorld!" are both string types.

int Integer

This is the second common data type we will introduce: integer, the English word for integer is integer, abbreviated as int. In the Python world, integers are actually the same as those defined in real-world mathematics: they are a general term for positive integers, negative integers, and zero, and are numbers without decimal points.

Unlike C and Java, Python integers have no length limit. In theory, they can approach ∞.

An integer is essentially a number without a decimal point. It can go without the quote jacket and directly team up with the print() function to output, for example, the following code:

The output result is 666

We already know that strings are protected by 【quotes】, allowing them to be freely combined with symbols and other text data, such as Chinese and English. However, integers have shed their 【quote】jacket and can only take action alone. Once they encounter symbols and other text data, such as Chinese and English, they will immediately report an error; if you give an integer a 【quote】jacket, its type will become a string (str), and it will not be able to participate in related operations.

float Floating-Point Number

The English name for a floating-point number is float, which is different from integers (int) and strings (str) in that floats do not have an abbreviation.

Floating-point numbers are very easy to identify. They have an extra decimal point (.) compared to integers, for example, the numbers in the following code are all floats:

Numbers like 【4.5】 and 【3.14】 are floating-point numbers.

Floats might make you a bit dizzy because, unlike the feature that integer operations always produce precise results, floating-point number operations in computers are not precise and have rounding errors. This means that calculations are represented by binary 0s and 1s, with carry-over every two digits, which leads to differences between our calculations and the program’s results.

Forced Type Conversion

Simply wrap the data or variable you want to convert with the corresponding data type.

It is important to note that both integers and floats can be converted to strings. Converting a float to an integer will result in a loss of precision, and converting an integer to a float will automatically add .0. A string consisting entirely of digits can theoretically be converted to an integer or a float, but there will be a loss of precision.
